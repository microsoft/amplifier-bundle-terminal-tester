# Support

## How to file issues and get help

This project uses [GitHub Issues](https://github.com/microsoft/amplifier-bundle-terminal-tester/issues) to track bugs and feature requests. Please search the existing issues before filing new issues to avoid duplicates. For new issues, file your bug or feature request as a new Issue.

For help and questions about using this project, please see the [README](README.md) and [docs/TROUBLESHOOTING.md](docs/TROUBLESHOOTING.md).

Support for this project is limited to the resources listed above.
